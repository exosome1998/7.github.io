PART 2: Understanding the CSS Box Model (15 minutes)

What is the Box Model?

Every HTML element is a box. The CSS Box Model describes how these boxes work.

The four parts:

  1. Content - Your text, images, etc.
  2. Padding - Space INSIDE the border (between content and border)
  3. Border - A line around the element
  4. Margin - Space OUTSIDE the border (between elements)

Step 1: See the Box Model in Action

1.1 Add this temporary code to your styles.css:

}

1.2 Save and look at your browser

You should see:

  • Light blue background (the content area)
  • Dark blue border around each h2
  • Space inside the border (padding)
  • Space outside the border (margin)

1.3 Now try changing the values:

    background-color: lightblue;

Understanding Spacing Values

You can set spacing for all sides at once, or individually:

All sides the same:

padding: 20px;  /* 20px on all sides */

Different for each side (Top, Right, Bottom, Left):

padding: 10px 20px 30px 40px;
/* Top: 10px, Right: 20px, Bottom: 30px, Left: 40px */

Top/Bottom and Left/Right:

padding: 20px 40px;
/* Top and Bottom: 20px, Left and Right: 40px */

Individual sides:

padding-top: 10px;
padding-right: 20px;
padding-bottom: 30px;
padding-left: 40px;

Same rules apply to margin and border!

PART 3: Using Classes and IDs (15 minutes)

What Are Classes and IDs?

So far, we've styled ALL elements of the same type:

  • ALL <h1> headings look the same
  • ALL <p> paragraphs look the same

But what if you want to style SPECIFIC elements differently?

That's where classes and IDs come in!

Classes vs IDs

CLASS:

  • Can be used on MULTIPLE elements
  • Use a dot (.) in CSS
  • Use for styling groups of elements

ID:

  • Can ONLY be used on ONE element per page
  • Use a hashtag (#) in CSS
  • Use for unique elements

Analogy:

  • Class = School uniform (many students wear it)
  • ID = Your student ID number (only you have it)

Step 1: Understanding Class Syntax

In HTML, add a class:

<p class="highlight">This paragraph is special!</p>

In CSS, style the class:

.highlight {
    background-color: yellow;
    font-weight: bold;
}

Notice the dot (.) before the class name in CSS!

Step 2: Create a Highlight Class

2.1 In your styles.css, add this code:

/* Highlight class for important text */
.highlight {
    background-color: #fff3cd;
    color: #856404;
    padding: 3px 8px;
    border-radius: 5px;
    font-weight: bold;
}

What do these properties do?

Property Value What It Does
background-color #fff3cd Light yellow background
color #856404 Dark orange text
padding 3px 8px Space inside (top/bottom: 3px, left/right: 8px)
border-radius 5px Rounds the corners
font-weight bold Makes text bold

What is <span>?

  • Invisible until you style it with CSS!

Step 1: Create a Header Section

Let's make your main heading stand out even more!

1.1 In index.html, wrap your <h1> in a special div:

New properties explained:

Property What It Does
background: linear-gradient() Creates a color gradient (blend between colors)
text-shadow Adds shadow to text (makes it pop!)
opacity Transparency (0 = invisible, 1 = solid)

Understanding Gradients:

linear-gradient(direction, color1, color2);
  • 135deg - Angle (0deg = up, 90deg = right, 135deg = diagonal)
  • #667eea 0% - First color at start
  • #764ba2 100% - Second color at end

New concepts:

:hover - A special selector that applies when you mouse over the element!

What this does:

  • When you hover over a card, it lifts up slightly!
  • The shadow gets bigger
  • It all animates smoothly!

PART 6: Responsive Design Basics (10 minutes)

What is Responsive Design?

Your webpage should look good on different screen sizes:

  • Desktop computers (wide)
  • Tablets (medium)
  • Phones (narrow)

Step 1: Add Mobile-Friendly Meta Tag

1.1 Check your index.html - you should already have this in <head>:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

If not, add it! This tells mobile browsers to show the page at the right size.

Step 2: Add a Media Query

Media queries change styles based on screen size!

2.1 Add this at the VERY BOTTOM of your styles.css:

/* Mobile responsive design */
@media (max-width: 768px) {
    body {
        padding: 10px;
    }

    .header h1 {
        font-size: 32px;
    }

    .card {
        padding: 20px;
    }

    .footer {
        padding: 20px;
    }
}

What does this mean?

  • @media (max-width: 768px) - "If screen is 768px wide or less..."
  • Everything inside { } only applies on small screens
  • Makes padding smaller, fonts smaller, etc.

2.2 Test it!

In your browser, press F12 to open Developer Tools

Click the device icon (looks like a phone) in the top-left

Now you can see how your site looks on different devices!
